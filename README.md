# Rocketseat - NLW eSports

## Required

Node Version: >=16.17.0.

## Steps

- Rename .env.sample to .env and fill the variables;
- Install the dependencies with `yarn;
- Run the migrations.

## Running

- Run the server with `yarn dev`, if you are in mobile, then run `yarn start`.
